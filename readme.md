## **Stručný popis tématu (3–5 vět)**

E-shop je internetový obchod, který umožňuje zákazníkům nakupovat produkty. Např.: oblečení(kalhoty, rukavice, boty, atd...)
Nabízí široký výběr stylů, velikostí a značek(adidas, nike, puma, atd...)
## **Cílová skupina**

Cílovou skupinou jsou zákazníci od 12 let až po dospělé všech věkových kategorií, kteří hledají kvalitní, moderní, a pohodlné oblečení. E-shop je pro každého, kdo preferuje jednoduchý a rychlý nákup online.
