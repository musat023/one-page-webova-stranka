## **Název projektu**

Restaurace 

## **Žák**

Musat Isik

## **Stručný popis tématu (3–5 vět)**

Restaurace je moderní gastronomické místo, které nabízí kvalitní jídla a příjemnou atmosféru pro návštěvníky všech věkových kategorií. Webová prezentace slouží k představení nabídky jídel a nápojů, informací o podniku a možnosti rezervace. Menu bude načítáno dynamicky pomocí AJAXu ze souboru JSON. Stránka je navržena tak, aby byla přehledná, rychlá a dostupná i na mobilních zařízeních.

## **Cílová skupina**

Cílovou skupinou jsou návštěvníci od studentů až po rodiny a dospělé, kteří hledají kvalitní stravování, moderní prostředí a snadný způsob, jak získat informace o restauraci. Web je vhodný pro každého, kdo preferuje rychlý přehled nabídky a jednoduchou online rezervaci.

## **Návrh obsahových sekcí**
1. O restauraci – základní představení podniku, jeho atmosféry, konceptu a nabízených služeb.
2. Menu – přehled jídel a nápojů, které se budou dynamicky načítat pomocí AJAXu ze souboru JSON.
3. Galerie – ukázky interiéru, jídla a prostředí restaurace.
4. Rezervace – jednoduchý formulář pro odeslání rezervace na backend bez obnovování stránky.
5. Kontakt a adresa – informace o otevírací době, adrese, sociálních sítích a možnostech dopravy.
⸻
## **Návrh designu**
• Moderní a čisté barevné ladění (kombinace bílé, tmavě šedé, béžové a jemných zemitých odstínů).
• Přehledná horní navigace, která se při scrollování zmenší nebo přichytí k hornímu okraji.
• Sekce „Menu“ zobrazena v podobě karet nebo seznamu s fotografií, názvem a krátkým popisem jídla.
• Rezervační formulář s hlavními tlačítky (Odeslat, Vymazat) a přehledným uspořádáním.
• Responzivní rozvržení — na mobilních zařízeních se sekce skládají pod sebe, navigace se zjednodušuje do hamburger menu.
⸻
## **Použité technologie**
• Frontend: HTML, CSS nebo Tailwind CSS, JavaScript
• Data: JSON soubory pro menu, galerii a otevírací dobu
• AJAX: načítání menu a odesílání rezervací bez obnovování stránky
• Backend: jednoduchý PHP nebo Node.js endpoint pro zpracování rezervací a poskytování dat
• Verzování: Git + GitHub (repozitář sdílen s vyučujícím)
⸻
## **čekávaný přínos / co se naučím**
• práce s JSON soubory a dynamickým načítáním dat
